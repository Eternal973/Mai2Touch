# Mai2Touch
使用 Arduino 制作的 mai2 触摸。
通信数据格式可参考 [Data.txt](https://github.com/Sucareto/Mai2Touch/blob/main/Data.txt)。

#### 待完成：

#### 使用方法：  
上传程序。  
打开设备管理器，设置 Arduino 的 COM 号，1P = COM3，2P = COM4。

#### 已测试开发板：   
- NodeMCU 1.0（ESP-12E + CH340）
